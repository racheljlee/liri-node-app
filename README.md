## APIs / NPMs used:
+ Twitter, Spotify, OMDB API

## Twitter NPM:

## Spotify NPM:
+ `node liri.js spotify-this-song "[song name]"` will load song title, artist(s), preview url, album name:
![alt text](https://i.gyazo.com/fb27d7226afb415b6a45a599c46538bc.png)

## OMDB API:
